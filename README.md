# spring-redis #

This repository demonstrates how to create a basic web application using Spring Framework and Redis.

### How do I get set up? ###

* Compile
 
    ```bash
    $ gradle clean build
    ```

* Run Server

    ```bash
    $ gradle bootRun
    ```

* Use  
    By default the server will serve at:
    > http://localhost:8080/
